# update packages
```
sudo dnf update
sudo dnf upgrade
```

# update ShellgeiBot-Image
```
docker pull theoldmoon0602/shellgeibot
```

# update HTTPS
```
sudo certbot certonly --standalone
sudo systemctl reload nginx
```
