# SHELLGEI ONLINE JUDGE
SHELLGEI ONLINE JUDGE is a shell one-liner playground: https://shellgei-online-judge.com/

シェル芸オンラインジャッジはシェル芸で問題を解いて遊ぶウェブアプリです。
