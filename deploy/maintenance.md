# Maintenance

## Update docker image file

```sh
docker pull theoldmoon0602/shellgeibot
```

## Update Ubuntu

```sh
sudo apt update
sudo apt upgrade
```

## Update Amazon Linux

```sh
sudo dnf update
sudo dnf upgrade
```

# sudo chmod 400 example.pem
